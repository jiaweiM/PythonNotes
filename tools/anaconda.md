## 命令

https://conda.io/docs/using/pkgs.html

|命令|说明|
|---|---|
|conda list|列出已安装的包|
|conda search beautiful-soup|搜索指定包是否安装|
|conda update biopython|更新包|
|conda search python|To list the versions of Python that are available to install|
